## Cómo ejecutar el proyecto

### Requisitos
- Unity 2022.x o superior (verificar versión en `ProjectSettings/ProjectVersion.txt`)
- Git instalado

### Pasos
1. Clonar el repositorio:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Abrir Unity Hub
3. Hacer clic en "Add" y seleccionar la carpeta del proyecto
4. Abrir el proyecto con la versión de Unity correspondiente
5. Abrir la escena principal en `Assets/Scenes/SampleScene.unity`
6. Pulsar Play para ejecutar

## Estructura de ramas
- `main` - Rama principal (solo versiones estables)
- `develop` - Rama de integración
- `feature/<nombre>` - Nuevas funcionalidades
